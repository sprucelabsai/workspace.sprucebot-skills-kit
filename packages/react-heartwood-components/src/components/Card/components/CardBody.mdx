# CardBody

> CardBody is the main content area of a Card.

<Playground>
	<CardBody>
		<p>
			Two roads diverged in a wood, and I took the one less traveled by, And
			that has made all the difference.
		</p>
	</CardBody>
</Playground>

## When to use

Only use a CardBody inside of a Card.

## When not to use

Don’t use a CardBody outside of a Card.
