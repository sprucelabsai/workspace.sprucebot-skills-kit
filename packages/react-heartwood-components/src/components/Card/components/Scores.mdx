# Scores

> Scores provide users with a quick heads-up on what’s happening.

## When to use

Use Scores inside of a Card to give a user a brief report on how things are going.

## When not to use

Don’t use Scores to display evergreen information or anything that a user can’t influence. Scores should never be used outside of a Card.
